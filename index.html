ENTERPRISE WEB APP

SQL Client GUI Application

A three-tier distributed web application demonstrating enterprise architecture with role-based access control

This is a project that demonstrates a three-tier distributed web application architecture using Java Servlets, JSP, and MySQL. The application simulates an enterprise system with role-based access control and database operations.

The application works as a multi-user database client where different types of users can interact with a MySQL database through a web interface. When a user logs in, the system authenticates them against a credentials database and routes them to their specific interface based on their role - whether they're a Root administrator, Client viewer, Data Entry clerk, or Accountant.

Each user type connects to the database with different MySQL accounts that have varying permission levels. Root users can execute any SQL command and see business logic in action, while Client users can only run SELECT queries. Data Entry users work through HTML forms to add suppliers, parts, jobs, and shipments, while Accountants have specialized interfaces for financial operations.

The most interesting feature is the automated business logic: when the Root user performs operations involving shipments with quantities of 100 or more, the system automatically detects this and increases the affected supplier's status by 5 points. This demonstrates how enterprise applications can have built-in business rules that execute automatically based on data changes.

Tech stack

Backend

  • Java Servlets using Jakarta EE framework
  • Apache Tomcat web container
  • MySQL with multiple schemas (project4, credentialsDB)
  • MySQL Connector/J 9.2.0 for database connectivity

Frontend

  • JSP (JavaServer Pages) for dynamic web content
  • Custom CSS with black backgrounds and colored headers
  • HTML Forms for structured data entry
  • Role-specific interfaces for different user types
WEB APPLICATION

Calendar Tracker

A modern, lightweight planner that helps students manage classes, events, and schedules in one place

This is a project that demonstrates a modern web application for student schedule management using React and modern CSS. The application combines a clean calendar interface with smart event management and customizable settings.

The application works as a comprehensive planner where students can manage their classes, events, and schedules through an interactive calendar interface. Users can view their month at a glance, add events quickly through simple forms, and organize everything by course to keep classwork separate and organized.

Each event can be categorized by course with different permission levels for viewing and editing. The system features color-coded entries, quick event previews, and an intuitive search function that instantly filters events and courses. Users can also import and export their data for backup or device transfers.

The most interesting feature is the responsive design with full dark mode support: the application automatically adapts to desktop and mobile screens while maintaining beautiful theme switching with carefully tuned contrast and colors. This demonstrates how modern web applications can provide seamless user experiences across different devices and preferences.

Tech stack

Frontend

  • React using Create React App framework
  • Modern CSS with CSS custom properties for theming
  • Responsive Design with mobile-first approach
  • Component Architecture for reusable UI elements

Features

  • Interactive Calendar with month view and clickable days
  • Event Management with add, edit, and remove functionality
  • Course Organization for academic event categorization
  • Dark Mode Toggle with accessible color schemes
